Règles firewall — Copier pour ACL

Remplacer VM_IP par l'adresse IP de la VM SavQonnect. Ces règles sont prêtes à être intégrées dans la configuration firewall.

Entrant (Inbound) — Accès à SavQonnect

Ports à autoriser vers la VM 
# SavQonnect - Règles ENTRANTES (INBOUND)
# Autoriser depuis réseaux UQAM / VPN

# API Backend (Production) - CRITIQUE
ALLOW TCP ANY → VM_IP:7070

# API Backend (Staging)
ALLOW TCP DEV_NETWORK → VM_IP:7071

# Frontend Nginx (si exposé)
ALLOW TCP ANY → VM_IP:80
ALLOW TCP ANY → VM_IP:443

# Monitoring (admins uniquement)
ALLOW TCP ADMIN_VLAN → VM_IP:3000   # Grafana
ALLOW TCP ADMIN_VLAN → VM_IP:9090   # Prometheus
ALLOW TCP ADMIN_VLAN → VM_IP:3100   # Loki

Sortant (Outbound) — Services cloud

Domaines à autoriser depuis la VM 
# SavQonnect - Règles SORTANTES CLOUD
# VM → Internet (HTTPS 443)

# Azure Entra ID - SSO UQAM (CRITIQUE)
ALLOW TCP VM_IP → login.microsoftonline.com:443
ALLOW TCP VM_IP → graph.microsoft.com:443

# Podio (Bons de travail)
ALLOW TCP VM_IP → api.podio.com:443

# Solstice Cloud (Mersive)
ALLOW TCP VM_IP → api.mersive.com:443
ALLOW TCP VM_IP → *.mersive.com:443

# Q-SYS Reflect
ALLOW TCP VM_IP → reflect.qsc.com:443

# LibCal (réservations)
ALLOW TCP VM_IP → uqam-ca.libcal.com:443

# Meraki Dashboard
ALLOW TCP VM_IP → api.meraki.com:443

# Sennheiser Cockpit (interne UQAM)
ALLOW TCP VM_IP → cockpit.uqam.ca:443

# DeepFreeze Cloud
ALLOW TCP VM_IP → *.faronics.com:443

# PostgreSQL cloud (optionnel)
ALLOW TCP VM_IP → *.neon.tech:5432

# Qdrant Cloud (RAG - optionnel)
ALLOW TCP VM_IP → *.qdrant.io:6334

Sortant — Équipements AV (VLAN audiovisuel)

Ports vers VLAN équipements 
# SavQonnect - Règles SORTANTES VLAN AV
# VM → VLAN_AV (équipements audiovisuels)

ALLOW TCP VM_IP → VLAN_AV:4352   # PJLink (projecteurs)
ALLOW TCP VM_IP → VLAN_AV:3629   # ESC/VP.net (Epson)
ALLOW TCP VM_IP → VLAN_AV:45     # Sennheiser SSCv1
ALLOW TCP VM_IP → VLAN_AV:443    # Sennheiser SSCv2, Mersive, DDM, VIA
ALLOW TCP VM_IP → VLAN_AV:5443   # Mersive OpenControl v2 (prioritaire)
ALLOW TCP VM_IP → VLAN_AV:1710   # Q-SYS QRC
ALLOW TCP VM_IP → VLAN_AV:22     # Crestron SSH
ALLOW TCP VM_IP → VLAN_AV:41795  # Crestron CTP (Toolbox)
ALLOW TCP VM_IP → VLAN_AV:80     # HTTP (Mersive, Epson, VIA)
ALLOW UDP VM_IP → VLAN_AV:9      # Wake-on-Lan (DeepFreeze)
ALLOW UDP VM_IP → VLAN_AV:161    # SNMP (discovery)
ALLOW TCP VM_IP → VLAN_AV:23     # Telnet (Sennheiser legacy)
